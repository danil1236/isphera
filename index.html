Справочник тегов.

Файлы с расширением .html открываются в браузере и состоят из тегов.

  • теги для текста
  • теги для выделения текста
  • теги для графических элементов
  • теги задающие структуру страницы

1. Теги для текста

  • p - абзац от paragraph
  • h1..h6 - заголоки от англ. heading
  • ul/ol - нумерованный и маркированный списки от англ. unordered/ordered list li - пункты списков от list item.
  • br - перенос строки от англ. break.
  • a - ссылка от anchor (якорь).
    • подробнее.

2. Теги для выделения текста.

  • b - жирный от англ. bold.
  • i - курсивный от англ. italic.
  • del - перечеркнутый от англ. delete.

3. Теги для графических элементов.

  • hr - линия от англ. horizontal rule.
  • img - картинка от англ. image.

4. Теги задающие структуру страницы.

  • !Doctype html - тег инструкция для браузера открывает html код.
  • html - заключает в себя все содержимое WEB страницы.
  • head - для зранения служебной информации для страницы.
  • body - Весь контент страницы.
  • section - Задает раздел документам.
  • div - Универсальный блок.
